## 📋 프로젝트 개요

해피패스 주식회사의 공식 B2B 브랜딩 웹사이트입니다. QA 검증 업무 대행 및 인력 파견 통합 서비스를 제공하는 전문 기업의 온라인 프레즌스를 구축하였습니다.

### 🎯 프로젝트 목표
- **B2B 신뢰도 구축**: 대기업 레퍼런스를 중심으로 전문성 강조
- **서비스 명확화**: QA 검증, 인력 파견, PM 리딩 서비스 명확한 전달
- **리드 생성**: 효과적인 문의 폼을 통한 비즈니스 기회 창출
- **브랜드 정체성**: 전문적이면서도 접근 가능한 브랜드 이미지 구축

## 🛠️ 개발 & 배포

### 로컬 실행
1. 저장소 클론 또는 파일 다운로드
2. `index.html` 파일을 브라우저에서 열기
3. 로컬 서버 실행 (권장):
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (http-server)
   npx http-server
   ```
4. 브라우저에서 `http://localhost:8000` 접속

### 배포
- **Publish 탭 사용**: 원클릭 배포
- **또는 수동 배포**:
  - Netlify, Vercel, GitHub Pages
  - AWS S3 + CloudFront
  - 기존 웹 호스팅 서비스

### 브라우저 호환성
- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+
- ✅ 모바일 브라우저 (iOS Safari, Chrome Mobile)
